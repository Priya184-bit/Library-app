# Library-app
# 📚 Online Library System

This is a React-based Online Library System built using Vite.

## 🚀 Features

- Browse books by category
- Search books
- View book details
- Add new books
- Redux state management

## 🛠️ Tech Stack

- React (Vite)
- React Router DOM
- Redux Toolkit

## ▶️ Run Project

npm install  
npm run dev
